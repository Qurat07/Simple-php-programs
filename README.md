# Simple-php-programs

Small collection of simple PHP example programs and exercises.  
This repo is intended for learners and anyone who wants quick, copy-pasteable PHP examples.

Badges
- CI: ![PHP CI](https://github.com/Qurat07/Simple-php-programs/actions/workflows/php-ci.yml/badge.svg)
- License: MIT

Overview
- Beginner-friendly PHP snippets and small programs.
- Each program is self-contained with instructions to run and a short explanation.
- Good for learning PHP fundamentals and for use in tutorials.

Contents
- /examples — each PHP example or mini-project
- /docs — optional documentation or demo GIFs
- /tests — PHPUnit tests (if present)

Quick start
1. Clone the repo:
   git clone https://github.com/Qurat07/Simple-php-programs.git
2. Run an example:
   php examples/hello-world.php

If examples rely on Composer, run:
   composer install
