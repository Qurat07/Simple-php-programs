# Contributor Covenant Code of Conduct

Our community is dedicated to providing a friendly, welcoming and harassment-free experience for everyone. We expect everyone participating in the project to follow this Code of Conduct.

- Be respectful and considerate in all communications.
- Refrain from demeaning, discriminatory or harassing behavior.
- When you disagree, assume good intent and try to resolve issues constructively.

Report unacceptable behavior by opening an issue or contacting the repository maintainers.

For the full text, see: https://www.contributor-covenant.org/version/2/0/code_of_conduct/
